<h3>
复制到系统剪切板<br>
<span class="gray">2017年11月18日20:59:18</span>
</h3>
<p>
通过命令 "+yy 即可将vim文件中的内容复制到系统剪切版中<br>
vim共用12个剪切版,分别为:0..9,a,"以及+.<br>
其中 " 为临时粘贴板,+为系统剪切板.<br>
"nyw(n copy word) -> 复制当前单次到n号剪切板<br>
"np(n paste) -> 粘贴n号剪切板内容到当前位置后<br>
"+yy / "+Y -> 复制当前行到系统剪切板<br>
"+ny -> 复制当前行加下面n行到系统剪切板<br>
"+p -> 粘贴系统剪切板内容到当前位置后
</p>

<h3>
必杀技<br>
<span class="gray">2017年11月19日10:56:57</span>
</h3>
<p>
0 -> 到行头<br>
 ^ -> 到本行第一个非空格的字符<br>
$ -> 到行尾<br>g_ -> 到本行最后一个不是空格的位置<br>
fn/tn -> 到本行下一个字母为n(任意字符)的位置,限于本行<br>
nfn/ntn -> 到本行第n次出现字符n的位置前<br>
Fn/Tn nFn/nTn -> 向前搜索,与f t方向相反<br>
f和t的区别在于:如fetch ft -> 光标会跳转到t上 tt -> 光标会跳转到e上<br>
dfn/dtn-> 用来删除本行内容,直到遇见n(任意字符)<br>

</p> 

<h3>
区域选择<br>
<span class="gray">2017年11月19日20:37:53</span>
</h3>
<p>
在区域选择中关键字分为a和i.a和i的区别类似于f与t.a会包含终止符号,i会选择终止符号前的内容.<br>
语法格式:命令(d,y,v等任何命令)a/i范围符号(范围符号可以是",),',},]等可以包含其他内容的符号)<br>
如:(array.rindex('Ruby')),光标在u字母上<br>
va' -> 光标选中'Ruby'<br>
vi' -> 光标选中Ruby<br>
va) -> 光标选中('Ruby')<br>
vi) -> 光标选中'Ruby'<br>
v2a) -> 光标选中(array.rindex('Ruby'))<br>
v2i) -> 光标选中array.rindex('Ruby')<br>

</p>
